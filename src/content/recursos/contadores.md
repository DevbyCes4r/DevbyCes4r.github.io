<h2 id="prompts-ia"><strong>⚡</strong> Prompts de IA para ser 3X más productivo</h2>

Copia y pega estos *prompts* (instrucciones) en herramientas como ChatGPT o Claude para facilitar tu trabajo:

### 1. Generación de informes financieros
```text
Actúa como un CFO senior con experiencia en [sector/industria].
Con base en los siguientes datos financieros correspondientes al periodo [mes/año]:
[PEGA AQUÍ LOS DATOS O RESUMEN CONTABLE]
1. Elabora un informe financiero ejecutivo profesional.
2. Analiza variaciones vs. [mes anterior / presupuesto / mismo periodo año anterior].
3. Identifica riesgos financieros relevantes.
4. Detecta oportunidades de optimización.
5. Incluye conclusiones estratégicas para la gerencia.
6. Presenta el informe en formato claro, listo para enviar a dirección.

Tono: ejecutivo, analítico y orientado a toma de decisiones.
```

### 2. Conciliaciones bancarias
```text
Actúa como auditor contable especializado en conciliaciones bancarias. Compara los siguientes datos:
Extracto bancario: [PEGA AQUÍ]
Libro auxiliar contable: [PEGA AQUÍ]

1. Identifica diferencias.
2. Clasifica cada diferencia (error contable, gasto no registrado, ingreso pendiente, comisión bancaria, etc.).
3. Sugiere ajustes contables necesarios.
4. Resume hallazgos en tabla estructurada.
5. Señala posibles riesgos o inconsistencias recurrentes.

Nivel de detalle: alto.
```

### 3. Proyección de flujo de caja
```text
Actúa como analista financiero experto en planeación y modelación financiera.
Con base en:
- Ingresos proyectados: [DATOS]
- Egresos fijos: [DATOS]
- Egresos variables: [DATOS]
- Deudas vigentes: [DATOS]
- Horizonte de análisis: [3/6/12 meses]

1. Proyecta el flujo de caja mensual.
2. Identifica posibles meses con déficit.
3. Propón 3 escenarios: conservador, probable y optimista.
4. Recomienda estrategias para mantener la liquidez.
5. Resume riesgos financieros clave.
Presenta resultados en formato claro y estructurado.
```

### 4. Análisis de costos y rentabilidad
```text
Actúa como controller financiero especializado en análisis de costos.
Información:
- Costo fijo mensual: [VALOR]
- Costo variable unitario: [VALOR]
- Precio de venta: [VALOR]
- Volumen promedio mensual: [VALOR]

1. Calcula margen de contribución.
2. Determina el punto de equilibrio.
3. Evalúa la rentabilidad actual.
4. Identifica oportunidades para mejorar margen.
5. Propón recomendaciones estratégicas.
Incluye explicación técnica clara.
```

### 5. Elaboración de presupuestos
```text
Actúa como especialista en planificación financiera empresarial.
Con base en el histórico de los últimos [X] años:
[PEGA AQUÍ DATOS HISTÓRICOS]

1. Diseña un presupuesto para el año [AÑO].
2. Proyecta crecimiento basado en tendencia.
3. Ajusta por inflación estimada de [X%].
4. Identifica áreas de sobrecosto.
5. Sugiere métricas de control presupuestal.
```

### 6. Análisis tributario y planificación fiscal
```text
Actúa como asesor tributario especializado en legislación de [PAÍS].
- Utilidad antes de impuestos: [VALOR]
- Deducciones aplicables: [DETALLE]
- Tipo de empresa: [SAS, LTDA, persona natural, etc.]

1. Estima el impuesto a pagar.
2. Identifica beneficios fiscales posibles.
3. Evalúa riesgos tributarios.
4. Sugiere estrategias de optimización fiscal legal.
5. Advierte posibles contingencias.
```

### 7. Análisis de indicadores financieros (KPIs)
```text
Actúa como analista financiero senior.
Datos:
- Activos totales: [VALOR]
- Pasivos totales: [VALOR]
- Patrimonio: [VALOR]
- Utilidad neta: [VALOR]
- EBITDA: [VALOR]

1. Calcula los principales indicadores financieros.
2. Interpreta cada indicador.
3. Compara contra estándares del sector [INDUSTRIA].
4. Evalúa la salud financiera.
5. Resume en conclusiones ejecutivas.
Formato: reporte ejecutivo para junta directiva.
```

### 8. Redacción de informes para inversionistas
```text
Actúa como director financiero preparando informe para inversionistas.
- Empresa: [NOMBRE]
- Sector: [INDUSTRIA]
- Resultados del periodo: [DATOS]
- Retos enfrentados: [DETALLE]

1. Redacta un informe claro y persuasivo.
2. Explica variaciones relevantes.
3. Destaca logros financieros.
4. Proyecta expectativas futuras.
5. Mantén tono profesional y estratégico.
Formato listo para presentación o mailing.
```

### 9. Automatización de clasificación contable
```text
Actúa como contador experto en clasificación contable bajo normativa [NIIF / COLGAAP / otra].
Clasifica los siguientes movimientos:
[LISTA DE MOVIMIENTOS]

1. Asigna cuenta contable adecuada.
2. Justifica clasificación.
3. Detecta posibles errores.
4. Sugiere mejora en registro contable.
Devuelve resultado en tabla estructurada.
```

### 10. Evaluación de viabilidad financiera de proyectos
```text
Actúa como analista de evaluación de proyectos de inversión.
- Inversión inicial: [VALOR]
- Flujos proyectados: [DETALLE]
- Tasa de descuento: [X%]
- Horizonte: [AÑOS]

1. Calcula VAN y TIR.
2. Determina periodo de recuperación.
3. Evalúa sensibilidad ante variaciones del [X%].
4. Emite recomendación fundamentada.
5. Resume riesgos financieros principales.
Nivel técnico: alto.
```
