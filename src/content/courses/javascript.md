## Sobre este curso

JavaScript es el lenguaje de programación más popular del mundo. Con él, podrás crear interactividad, animaciones complejas y aplicaciones web completas.

### Contenido del Curso
- **Fundamentos**: Variables, tipos de datos, operadores.
- **Lógica de Control**: Condicionales, bucles y funciones.
- **DOM Manipulation**: Modificar el HTML y CSS desde código.
- **Eventos**: Responder a clicks, inputs y acciones del usuario.
- **ES6+**: Las características modernas del lenguaje (Arrow functions, Destructuring, Promises).

### ¿Por qué JavaScript?
Es el único lenguaje que corre nativamente en el navegador. Es esencial para frontend (React, Vue, Angular) y backend (Node.js).
