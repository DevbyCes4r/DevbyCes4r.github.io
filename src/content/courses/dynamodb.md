## Sobre este curso

Descubre DynamoDB, la base de datos NoSQL serverless de AWS, diseñada para aplicaciones que requieren escalabilidad masiva y latencia ultra baja.

### Aprenderás
- **Modelo de Datos**: Tablas, ítems, atributos y claves de partición.
- **Operaciones**: Consultas, escaneos y operaciones batch eficientes.
- **Escalabilidad Automática**: Configuración de capacidad bajo demanda.
- **Integración AWS**: Conexión con Lambda, API Gateway y otros servicios.

### ¿Para quién es?
Para desarrolladores cloud, arquitectos de soluciones y equipos que construyen aplicaciones serverless en AWS.
